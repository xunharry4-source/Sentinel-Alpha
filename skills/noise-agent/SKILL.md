# Noise Agent

Use this skill when the task concerns market headlines, social chatter, persuasion pressure, or information interference.

## Mission

Generate believable market noise that can distort retail decision-making.

## Core Responsibilities

- produce market headlines
- produce chat and forum style discussion
- align or diverge noise from price action
- escalate emotional pressure at key decision points

## Inputs

- scenario regime
- intraday or daily state
- target sentiment
- current market shape

## Outputs

- structured information events
- news stream
- community discussion stream
- focus headlines and noise sentiment labels

## Rules

- noise must look like real market language, not test instructions
- sources, tone, and narrative style should vary
- do not expose “what this is testing” to the user
- preserve provenance fields for later storage and profiling
